# 人力管理系统

> **前言**
>
> 此系统是一个PC端的后台管理系统
>
> 技术栈：Vue2全家桶 + axios + Element UI
>
> 后端接口可能会失效。  

> **运行环境**
>
> 克隆源码➡package.json所属文件下npm i下载依赖，之后npm run dev运行，node18以上运行npm run dev_t
>
> 在http://localhost:8888访问项目
